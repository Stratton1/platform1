# Platform Builder

## Project Structure

Main application directory: `platformbuilder/`

### Key Directories
- `src/pages/`: Application routes
- `src/components/`: Reusable UI components
- `public/`: Static assets

## Configuration
- Using Next.js 15.0.2
- Tailwind CSS for styling
- PostCSS for processing
